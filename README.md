# Selected-Code-Examples

Selected examples of code for job applications. Below is a breif description of each of the included projects
* Python App - This is a work in progress application I am developing for my current employer. The goal is to make a tool that lets us easily create figures in our own brand style. This is my first signifcant Python project, and I plan on updating it here as I make additions. 
* Inequality Model - Code used to study inequality in Ohio. This project was designed to be made up of flexible modules that could be repurposed for future policy analysis projects
* IPUMS Data collection and cleaning - Code used for downloading data via the IPUMS API. Cleaning script is specific for a minimum wage research project. 
* Plan B - Code for my Plan B research project. The project was an analysis of survey data collected by the Women in Neurology Collaberative Study
* Causal Inference Code - The final exam for my causal inference course (PUBH 7485). Data come from the National Longitudinal Mortality Study.
* Teaching Code - Code examples from a lecture I gave as a TA for STAT 4893W, Consultation and Communication for Statisticians with Dr. Kuzmak. The goal of this lecture was to give students a first look at SQL and Python and their basic usage.
